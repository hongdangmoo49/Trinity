# 작업 상세 인수 기준 라벨 일관화

## 배경

한국어 UI에서 blueprint acceptance criteria는 중앙 설계안과 리포트에서 `인수 기준`으로 표시된다. 하지만 작업 패키지 상세 모달만 `수락 기준`을 사용해 같은 개념이 화면마다 다르게 보인다.

## 목표

- 작업 패키지 상세 모달의 `acceptance_criteria` 한국어 라벨을 `인수 기준`으로 통일한다.
- 영어 라벨과 작업 패키지 상세 렌더링 구조는 변경하지 않는다.
- 관련 한국어 상세 모달 테스트를 보강한다.

## 작업 범위

1. `WorkPackageDetailModal`의 한국어 `acceptance_criteria` 라벨을 수정한다.
2. 한국어 상세 모달 테스트에 `## 인수 기준` 검증을 추가한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- acceptance criteria 데이터 구조 변경.
- 영어 상세 모달 문구 변경.
- 중앙 설계안/리포트 렌더링 변경.
