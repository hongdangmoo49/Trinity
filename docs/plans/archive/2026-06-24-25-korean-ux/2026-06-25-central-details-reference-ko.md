# 중앙 패널 상세 안내문 한국어화

## 배경

한국어 중앙 패널의 작업 패키지 안내문에 `Inspector 또는 Report`라는 영어 UI 명칭이 남아 있다. 같은 앱의 다른 한국어 화면에서는 `인스펙터`, `리포트` 표현을 쓰고 있으므로 중앙 패널도 같은 용어를 사용하는 편이 자연스럽다.

## 목표

- 중앙 패널의 상세 안내문에서 `Inspector 또는 Report`를 `인스펙터 또는 리포트`로 변경한다.
- 영어 안내문과 중앙 패널 렌더링 구조는 변경하지 않는다.
- 관련 label 테스트를 보강한다.

## 작업 범위

1. `CentralAgentView` 한국어 `details_in_inspector` 라벨을 수정한다.
2. 중앙 패널 한국어 label 테스트를 추가한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 인스펙터/리포트 화면 동작 변경.
- work package overview 렌더링 구조 변경.
- 영어 문구 변경.
