# 리포트 Risk 라벨 일관화

## 배경

한국어 UI의 실행 매트릭스, 작업 상세 모달, 중앙 액션은 risk를 `리스크`로 표시한다. 그러나 Textual 리포트 화면 필드 라벨만 `위험`을 사용해 같은 개념이 화면마다 다르게 보인다.

## 목표

- Textual 리포트 화면의 `Risk`, `Risks` 한국어 라벨을 `리스크`로 맞춘다.
- 영어 라벨과 리스크 값 렌더링은 변경하지 않는다.
- 관련 라벨 테스트를 보강한다.

## 작업 범위

1. `REPORT_FIELD_LABELS_KO`의 `Risk`, `Risks` 값을 수정한다.
2. 리포트 presenter 테스트에서 field label 기대값을 검증한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 리스크 값(`높음`, `보통`, `알 수 없음`) 표시 로직 변경.
- 마크다운 export 라벨 변경.
- 영어 UI 변경.
