# 설정 값 표시명 한국어화

## 배경

설정 화면의 한국어 모드에서는 주요 라벨이 한국어로 표시되지만, 선택 값과 미리보기 값은 `system`, `comfortable`, `normal`처럼 내부 설정 키가 그대로 노출된다. 사용자는 저장되는 키보다 현재 화면 상태를 먼저 읽기 때문에, 한국어 화면에서는 값도 읽기 쉬운 표시명으로 보여주는 편이 자연스럽다.

## 목표

- 한국어 설정 화면의 드롭다운 선택지에 언어별 표시명을 적용한다.
- 미리보기의 테마 모드, 밀도, 중앙 프로바이더/모델, 에이전트 기본 모델 표시에도 같은 표시명을 사용한다.
- 저장되는 설정 값, provider id, model id는 변경하지 않는다.
- 영어 화면은 기존 표시와 호환되도록 원래 값 중심의 표현을 유지한다.

## 작업 범위

1. `SettingsScreen._display_value()`가 설정 값 표시명을 언어별 라벨로 반환하도록 확장한다.
2. `_preview_text()`에서 raw 설정 값과 중앙 모델 프리셋 대신 표시명을 사용한다.
3. 한국어 설정 미리보기 테스트에서 값 표시명이 한국어로 렌더링되는지 확인한다.
4. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 실제 config 파일에 저장되는 값 변경.
- provider/model id의 번역 또는 별도 alias 도입.
- 설정 화면 레이아웃 변경.
