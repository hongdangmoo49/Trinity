## 5. 남은 검증

자동 테스트는 unit/mock 중심이다. v0.7.0 완료 전에는 WSL/tmux 실환경 smoke가 필요하다.

필수 smoke:

1. Claude/Codex/Gemini 모두 인증 완료 상태에서 설계 요청.
2. Gemini만 auth required 상태에서 strict mode 중단 확인.
3. degraded mode에서 ready agent만으로 workflow 진행 확인.
4. 사용자 의사결정 질문 생성, 답변, 재개 확인.
5. execution package dispatch와 `Task Results` 기록 확인.
6. context threshold 근처에서 deliberation/execution 사이 rotation 확인.

## 6. 알려진 경고
