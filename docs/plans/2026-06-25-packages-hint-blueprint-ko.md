# 패키지 힌트 blueprint 용어 한국어화

## 배경

한국어 `/packages` 결과에서 작업 패키지가 없을 때 표시하는 안내문이 `blueprint 또는 로컬 WP 그래프...`로 시작한다. 같은 파일의 다른 실행 메시지는 이미 `설계안`을 사용하므로, 한국어 UI에서는 동일한 개념을 같은 용어로 표시하는 편이 자연스럽다.

## 목표

- 한국어 패키지 안내문에서 `blueprint`를 `설계안`으로 바꾼다.
- `WP 그래프`처럼 프로젝트에서 이미 쓰는 축약 용어는 유지한다.
- 영어 안내문은 변경하지 않는다.

## 작업 범위

1. `STATUS_CONTEXT_LABELS["ko"]["packages_hint"]` 문구를 수정한다.
2. `/packages` 관련 한국어 테스트 기대값을 갱신한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 영어 `/packages` 안내문 변경.
- 작업 패키지 생성/표시 로직 변경.
- `WP` 축약어 번역.
