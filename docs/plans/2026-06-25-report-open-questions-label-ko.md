# 리포트 Open Questions 섹션명 일관화

## 배경

한국어 리포트 화면의 `Open Questions` 섹션은 `열린 질문`으로 표시되지만, 마크다운 리포트 내보내기에서는 같은 섹션을 `미해결 질문`으로 표시한다. 사용자 입장에서는 같은 리포트 정보가 화면과 내보내기에서 다른 용어로 보이는 문제가 있다.

## 목표

- Textual 리포트 화면의 `Open Questions` 한국어 섹션명을 `미해결 질문`으로 맞춘다.
- 마크다운 export의 기존 `미해결 질문` 표현과 일치시킨다.
- 영어 섹션명과 질문 렌더링 로직은 변경하지 않는다.

## 작업 범위

1. `REPORT_SECTION_LABELS_KO["Open Questions"]` 값을 수정한다.
2. 리포트 화면 한국어 본문 테스트 기대값을 갱신한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 질문 데이터 구조나 렌더링 순서 변경.
- 마크다운 export 로직 변경.
- 영어 섹션명 변경.
