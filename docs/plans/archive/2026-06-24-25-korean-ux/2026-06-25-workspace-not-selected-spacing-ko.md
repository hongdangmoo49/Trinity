# 워크스페이스 미선택 문구 일관화

## 배경

한국어 UI에서 같은 상태를 가리키는 문구가 화면마다 다르게 표시된다. 실행 매트릭스와 일부 검증은 `선택 안 됨`을 쓰지만, 시작 화면과 Nexus 화면의 워크스페이스 라벨은 `선택 안됨`으로 표시된다.

## 목표

- 시작 화면과 Nexus 화면의 워크스페이스 미선택 문구를 `작업 폴더: 선택 안 됨`으로 통일한다.
- 기존 선택된 워크스페이스 표시 형식은 유지한다.
- 영어 문구는 변경하지 않는다.

## 작업 범위

1. `StartScreen` 한국어 라벨의 `workspace_not_selected` 문구를 수정한다.
2. `NexusScreen` 한국어 라벨의 `workspace_not_selected` 문구를 수정한다.
3. 관련 테스트 기대값을 갱신한다.
4. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 워크스페이스 선택 UX 또는 경로 표시 레이아웃 변경.
- 영어 문구 변경.
