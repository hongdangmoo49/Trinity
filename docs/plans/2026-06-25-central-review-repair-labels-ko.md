# 중앙 액션 review-repair 용어 일관화

## 배경

review-repair 관련 결과 메시지와 상세 패널은 `리뷰 보정`으로 통일되어 있지만, Nexus 중앙 액션 영역의 한국어 라벨과 tooltip에는 `리뷰 수리`가 남아 있다. 중앙 액션은 사용자가 보정 루프에서 직접 결정을 내리는 영역이므로, 같은 기능을 같은 용어로 표시해야 한다.

## 목표

- 중앙 액션 제목과 tooltip의 `리뷰 수리` 표현을 `리뷰 보정`으로 바꾼다.
- 기존 버튼 라벨과 action id는 유지한다.
- 영어 중앙 액션 문구는 변경하지 않는다.

## 작업 범위

1. `CentralAgentView` 한국어 label 사전의 review-repair 라벨/tooltip을 수정한다.
2. 한국어 중앙 액션 label 테스트를 추가한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- 중앙 액션 표시 조건 변경.
- workflow repair/recovery 로직 변경.
- 영어 문구 변경.
