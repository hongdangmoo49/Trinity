# 설정 미리보기 프로필/계약 라벨 한국어화

## 배경

설정 화면의 한국어 미리보기는 `테마 모드`, `밀도`, `중앙` 같은 상위 라벨은 한국어로 표시하지만, 에이전트 프로필 요약에는 `profile balanced`, `contracts ...` 같은 영어 라벨이 남아 있다. 설정 화면은 사용자가 구성 결과를 빠르게 확인하는 곳이므로, 언어 설정이 한국어일 때 고정 라벨도 일관되게 한국어로 보여야 한다.

## 목표

- 한국어 설정 미리보기에서 프로필 요약 라벨을 한국어로 표시한다.
- 출력 계약 요약의 고정 라벨을 한국어로 표시한다.
- 실제 profile strength 이름, workflow mode, output contract id는 구성 값이므로 원문을 유지한다.
- 영어 UI의 기존 표현은 유지한다.

## 작업 범위

1. `SettingsScreen`의 프로필/계약 요약 헬퍼가 화면 언어 라벨을 사용할 수 있도록 조정한다.
2. 한국어/영어 라벨 사전에 `profile`, `balanced`, `contracts`, `default`를 추가한다.
3. 한국어 설정 미리보기 테스트에 출력 계약 라벨 검증을 추가한다.
4. 패치 버전을 갱신하고 전체 테스트를 실행한다.

## 비범위

- profile strength 이름이나 output contract id 자체의 번역은 하지 않는다.
- 설정 화면 레이아웃이나 저장 흐름은 변경하지 않는다.
