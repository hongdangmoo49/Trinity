# 마크다운 리포트 owner 라벨 일관화

## 배경

한국어 UI에서 work package의 `owner_agent` 필드는 실행 매트릭스, 작업 상세 모달, Textual 리포트 화면에서 `소유자`로 표시된다. 하지만 마크다운 리포트 export에서는 같은 필드를 `담당`으로 표시한다. 같은 데이터가 화면과 내보내기에서 다른 라벨로 보이면 사용자가 비교하기 어렵다.

## 목표

- 마크다운 리포트 export의 `owner` 한국어 라벨을 `소유자`로 맞춘다.
- 기존 영어 라벨과 데이터 렌더링 순서는 변경하지 않는다.
- 관련 테스트 기대값을 갱신한다.

## 작업 범위

1. `REPORT_MARKDOWN_LABELS["ko"]["owner"]` 값을 수정한다.
2. 마크다운 리포트 한국어 테스트 기대값을 수정한다.
3. 패치 버전을 갱신하고 테스트를 실행한다.

## 비범위

- `owner_agent` 데이터 구조 변경.
- 화면 레이아웃 변경.
- 영어 리포트 라벨 변경.
